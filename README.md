
# Project Title

Rest assured E2E Framework.


# Prerequiste
JDK, IDE




## Features

- Logging
- Extent Report
- API Chaining
- E2E execution using pom.xml/testng.xml
- Moduler framework structure
- Dynamic Json body





## Running Tests

To run tests, run the following command

```bash
  mvn clean test
```
